## Getting Started

### Prerequisites

- Node.js (v14 or higher)
- npm or yarn

### Installation

1. Install dependencies:
```bash
npm install
```

2. Start the development server:
```bash
npm start
```

3. Open [http://localhost:3000](http://localhost:3000) to view it in the browser.

## Technologies Used

- React 18.2.0
- React Router DOM 6.20.0
- CSS3 (with modern features)
- Google Fonts (Inter & Playfair Display)

## Sections

1. **Header** - Fixed navigation with mobile menu
2. **Hero** - Eye-catching hero section with call-to-action
3. **About** - Information about the company and services
4. **Services** - Available 24/7 service section
5. **Stats** - Statistics and achievements
6. **Why Choose Us** - Service list and benefits
7. **Popular Services** - Featured therapy services
8. **Team** - Meet the psychiatrist team
9. **Testimonials** - Client testimonials with slider
10. **Blog** - Latest blog posts
11. **Appointment** - Booking form
12. **Footer** - Links and contact information

## Customization

All components are modular and can be easily customized. Update the dummy data in each component file to add your own content.

## Build for Production

```bash
npm run build
```

This creates an optimized production build in the `build` folder.
